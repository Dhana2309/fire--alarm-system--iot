# Fire Alarm System using IoT

This project is an IoT-based fire alarm system designed to detect fire and alert users in real-time.

## Features
- Detects fire using flame/gas sensors
- Real-time alert system
- Easy to use

## Technologies Used
- Arduino / ESP
- Sensors
- C/C++

## How It Works
The sensor monitors the environment continuously. When fire is detected, an alert is triggered.
